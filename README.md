🌍 Project Overview

Cyber attacks generate massive amounts of network data, making it difficult for security teams to quickly identify patterns and threats.

This project builds an interactive Cyber Threat Intelligence Dashboard that analyzes 40,000 network attack records to detect:

• attack patterns
• anomaly behavior
• vulnerable network segments
• effectiveness of defense systems

The goal is to simulate how a Security Operations Center (SOC) monitors cyber threats.

🎯 Project Objectives

The primary goal of this project is to build a data-driven cyber threat intelligence dashboard capable of:

🔍 Identifying cyber attack patterns
📈 Monitoring network attack trends
🧠 Detecting anomalies in network traffic
🌐 Understanding geographic attack distribution
🛡️ Evaluating security defense effectiveness

📊 Dashboard Architecture

The project consists of 5 analytical dashboards:

1️⃣ Threat Intelligence Summary

Overview of total attacks, anomaly scores, and severity levels.

2️⃣ Protocol & Network Traffic Analysis

Analyzes protocol usage, traffic types, and network segments.

3️⃣ Cyber Attack Behaviour Analytics

Examines attack trends and anomaly patterns over time.

4️⃣ Threat Intelligence Analytics

Identifies attack signatures and alert patterns.

5️⃣ Cyber Defense & Prevention

Evaluates security responses such as firewall logs and actions taken.
